# Image Gallery

A simple Image Gallery built using **HTML, CSS, and JavaScript** where clicking on thumbnail images dynamically updates the main displayed image.

## Features
- Main image preview
- Clickable thumbnail images
- Smooth image switching
- Hover effect on thumbnails
- Clean and centered UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
- The main image is displayed at the top of the gallery.
- All thumbnails are selected using `querySelectorAll()`.
- Clicking a thumbnail updates the `src` of the main image dynamically.
- CSS handles layout, centering, and hover effects.

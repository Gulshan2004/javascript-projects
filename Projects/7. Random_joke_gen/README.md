# Random Joke Generator

A simple Random Joke Generator built using **HTML, CSS, and JavaScript** that fetches jokes from a public API and displays them dynamically.

## Features
- Fetch jokes from an external API
- Async/Await implementation
- Error handling
- Loading message while fetching data
- Clean and centered UI

## Technologies Used
- HTML
- CSS
- JavaScript
- JokeAPI (Public API)

## How It Works
- When the button is clicked, JavaScript sends a request to a public Joke API.
- The response is converted into JSON using `res.json()`.
- The joke text is extracted and displayed dynamically in the DOM.
- Errors are handled using try/catch.

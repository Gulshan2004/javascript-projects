# Quiz App

An interactive Quiz App built using **HTML, CSS, and JavaScript** that allows users to answer multiple-choice questions with a timer and view their final score.

## Features
- Multiple-choice questions
- Countdown timer for each question
- Automatic progression when time runs out
- Correct and wrong answer highlighting
- Score display at the end
- Restart quiz functionality
- Dynamic question rendering

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
1. Questions are stored in a JavaScript array.
2. Each question and its options are rendered dynamically.
3. A timer starts for every question.
4. Selecting an answer highlights correct and incorrect choices.
5. The score is updated based on correct answers.
6. At the end, the final score is displayed and the user can restart the quiz.
